### 🧱 Getting  Started
  - First of all clone the project to your device 
       - `git clone https://github.com/liildev/portfolio.git`

    * Download the project
        - `npm i`
    * You can run the project in developer Mode.
        - `npm run dev`
    * Steps to compile the project
        - `npm run generate`
        - `npm run build`
